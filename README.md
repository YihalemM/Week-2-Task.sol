This project contains a simple Solidity smart contract created for GDG AASTU Week 2 Practice.
The contract demonstrates basic concepts of Ethereum smart contract development.

Features

Store and update a number

Read a private message using a view function

Add two numbers using a pure function

Toggle a boolean state

Check if a number is greater than 10

Functions

setNumber(uint newNumber)-updates the number variable

getMessage()-returns the private message

add(uint a, uint b) - returns the sum of two numbers

toggleActive()- flips the isActive boolean

isNumberBig()- returns true if number > 10

Requirements

Solidity version 0.8.x

Runs on Remix, Foundry, Hardhat, or any EVM-compatible environment
